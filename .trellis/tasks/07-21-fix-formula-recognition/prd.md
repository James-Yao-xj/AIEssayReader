# 修复PDF公式识别乱码并支持LaTeX渲染

## Goal

修复 PDF 文本提取中公式出现乱码/跳行的问题，并为中栏文本添加 LaTeX 渲染能力。

## Requirements

1. **改进文本提取算法**：用字体高度信息替代固定像素阈值（EPS=2px）判断"真换行"与"公式内位移"，同行内按 X 坐标排序保证阅读顺序
2. **中栏支持 KaTeX 渲染**：复用 `render.js` 的 marked+KaTeX 管线，中栏全文走 markdown 渲染（纯文本不受影响，`$...$` / `$$...$$` 公式自动渲染），无需切换按钮
3. **不做数学自动检测**：不尝试把纯 Unicode 数学符号自动包裹为 `$...$`，避免误判
4. **保持向后兼容**：纯文本 PDF 不引入退化；追问功能不受影响

## Acceptance Criteria

- [ ] 含公式的 PDF 提取后，公式不再出现乱码/跳行（同行公式字符保持在同一行）
- [ ] 中栏文本中若含 `$...$` / `$$...$$` 标记，公式被 KaTeX 正确渲染
- [ ] 纯文本 PDF 提取结果不退化（与修复前一致或更好）
- [ ] 中栏追问功能（文本选中 → "追问"按钮）不受影响，选中的是渲染后的可见文本
- [ ] `npm run build` 构建成功

## Out of Scope

- 数学自动检测（Unicode → LaTeX 转换）
- 扫描件 OCR
- 左栏 PDF 渲染改动

## Decisions

| 决策 | 选择 |
|------|------|
| 中栏渲染模式 | 全文 marked+KaTeX，无需切换按钮 |
| 数学自动检测 | 不做 |
